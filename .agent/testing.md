# Testing Requirements

## LWC Jest Tests
Required for:
- All @api methods
- All event handlers
- Error scenarios
- Conditional rendering logic

Mock required:
- @wire adapters
- Apex method calls
- Navigation
- Toast notifications

## Manual Testing Checklist
- [ ] Desktop Chrome
- [ ] Mobile Safari (or DevTools mobile view)
- [ ] Different user profiles
- [ ] With/without data
- [ ] Error scenarios
